# API 接口更新 - 随机相册

## 新增接口

### GET /api/v1/galleries/random

获取一个随机可见相册（基本信息 + 第一张照片 URL），自动过滤无照片的相册。

**响应 200**

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": "018f...",
    "title": "日常随拍",
    "cover": "https://cdn.example.com/cover.jpg",
    "photo_count": 12,
    "tags": ["日常", "生活"],
    "visible": true,
    "created_at": "2026-04-01T00:00:00Z",
    "updated_at": "2026-04-03T00:00:00Z",
    "first_photo_url": "https://cdn.example.com/photo1.jpg"
  }
}
```

> - 仅返回 `photo_count > 0` 的相册
> - `first_photo_url` 为该相册 z_index 最小的照片 URL

**错误响应**

```json
// 无可见相册或有照片的相册时
{ "code": 10001, "message": "资源不存在", "data": null }
```

---

## 路由顺序注意

该接口 `/api/v1/galleries/random` 在路由注册时需位于 `/api/v1/galleries/{id}` 之前，避免 "random" 被当作 id 参数匹配。
