# API 接口更新 - 2026/04/07

## 照片布局更新接口修复

### PATCH /api/v1/admin/galleries/{gallery_id}/photos
### PATCH /api/v1/admin/photos/{id}

**修复内容：** 部分更新时未传入的字段现在正确保留原值，不再被设为默认值（0）

修复前：
```json
// 只传 x，y/rotation/z_index 会被错误地设为 0
{ "id": "xxx", "x": 100 }
```

修复后：
```json
// 只传 x，y/rotation/z_index 保留原值
{ "id": "xxx", "x": 100 }
```

## 番剧删除接口变更

### DELETE /api/v1/admin/bangumi-info/:id

**变更说明：** 删除番剧时，现在会同时软删除关联数据

- 删除番剧信息 (`bangumi_info`)
- 同时删除关联的追番记录 (`watchlist`)
- 同时删除关联的评论 (`comments`，target_type='bangumi')

## 定时任务

### 每日凌晨 3 点执行

新增以下清理任务：

1. **孤儿文件清理** - 上传超过24小时但数据库无引用的文件
2. **软删除数据清理** - 清理超过 30 天的已删除记录
3. **孤儿关联记录清理** - 清理关联父记录已不存在的数据：
   - `social_links` - user_id 不存在于 users
   - `user_settings` - user_id 不存在于 users
   - `jwt_blacklist` - admin_user_id 不存在于 admin_users
   - `admin_permissions` - admin_user_id 不存在于 admin_users
   - `watchlist` - bangumi_id 不存在于 bangumi_info
   - `comments` - target_id 不存在（gallery/bangumi）
   - `comments` - reply_to_user_id 不存在于 users
   - `photos` - gallery_id 不存在于 galleries（跨库处理）
