## 相册查看页
该页面是单独布局结构，不使用通用布局
说明：
1. 图片可以倾斜一定角度
2. 图片可以重叠
3. 图片可以拖动
4. 图片点击放大转正角度显示在前景
5. 白色底片，图片内嵌略小
6. 不要圆角，类似真实照片
