## 一、链接模块 `/links`

前端已实现：`src/components/links/LinksSection.vue`

**说明**：链接页用于收藏常用网站链接，类似导航书签。

### 1.1 数据模型

```typescript
interface Link {
  id: string                   // UUIDv7
  title: string                // 网站名称
  url: string                  // 链接地址
  description: string          // 网站描述
  tags: string[]               // 标签数组（用于分组）
  icon: string | null          // 网站图标URL（可选）
  visible: boolean             // 是否可见
  sort_order: number           // 排序权重
  created_at: string           // 创建时间 ISO 8601
  updated_at: string           // 更新时间 ISO 8601
}
```

### 1.2 公开接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/links` | GET | 获取链接列表（分页） |
| `/api/v1/links/:id` | GET | 获取单个链接详情 |

#### 获取链接列表

```
GET /api/v1/links?page=1&per_page=10
GET /api/v1/links?tag=开发
```

**查询参数**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `page` | integer | 1 | 页码 |
| `per_page` | integer | 10 | 每页数量（最大50） |
| `tag` | string | - | 按标签筛选 |

**响应示例**

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "items": [
      {
        "id": "018f...",
        "title": "GitHub",
        "url": "https://github.com",
        "description": "全球最大的代码托管平台",
        "tags": ["开发"],
        "icon": null,
        "visible": true,
        "sort_order": 0,
        "created_at": "2026-04-01T00:00:00Z",
        "updated_at": "2026-04-01T00:00:00Z"
      }
    ],
    "pagination": { "page": 1, "per_page": 10, "total": 10, "total_pages": 1 }
  }
}
```

### 1.3 管理接口（需管理员权限）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/admin/links` | POST | 创建链接 |
| `/api/v1/admin/links/:id` | PATCH | 更新链接 |
| `/api/v1/admin/links/:id` | DELETE | 删除链接 |
| `/api/v1/admin/links/:id/visibility` | PATCH | 切换可见性 |

#### 创建链接

```
POST /api/v1/admin/links
Authorization: Bearer <token>
```

**请求体**

```json
{
  "title": "GitHub",
  "url": "https://github.com",
  "description": "全球最大的代码托管平台",
  "tags": ["开发", "代码"],
  "sort_order": 0
}
```

#### 更新链接

```
PATCH /api/v1/admin/links/:id
```

**请求体**

```json
{
  "title": "新名称",
  "description": "新描述",
  "tags": ["新标签"]
}
```

#### 删除链接

```
DELETE /api/v1/admin/links/:id
```

响应：`204 No Content`

## 三、友人帐模块 `/friends` ⚠️ 前端未实现

**说明**：友人帐是友链（朋友链接），支持申请添加功能。

### 3.1 数据模型

```typescript
interface Friend {
  id: string                   // UUIDv7
  name: string                 // 网站名称
  url: string                  // 网站链接
  avatar: string               // 头像/图标URL
  description: string          // 网站描述
  status: 'active' | 'pending' | 'inactive'  // 状态
  visible: boolean             // 是否显示
  sort_order: number           // 排序权重
  created_at: string
  updated_at: string
}
```

### 3.2 公开接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/friends` | GET | 获取友链列表 |

### 3.3 申请接口（需登录）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/friends/apply` | POST | 申请添加友链 |

**请求体**

```json
{
  "name": "我的博客",
  "url": "https://myblog.example.com",
  "avatar": "https://myblog.example.com/avatar.jpg",
  "description": "个人技术博客"
}
```

### 3.4 管理接口（需管理员权限）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/admin/friends` | GET | 获取友链列表（含申请） |
| `/api/v1/admin/friends` | POST | 添加友链 |
| `/api/v1/admin/friends/:id` | PATCH | 更新友链 |
| `/api/v1/admin/friends/:id` | DELETE | 删除友链 |
| `/api/v1/admin/friends/:id/approve` | POST | 批准申请 |
| `/api/v1/admin/friends/:id/reject` | POST | 拒绝申请 |

## 四、状态说明

### 番剧状态

| 状态 | 中文 | 说明 |
|------|------|------|
| `watching` | 在看 | 正在观看 |
| `want_to_watch` | 想看 | 想看但未开始 |
| `watched` | 看过 | 已看完 |
| `dropped` | 抛弃 | 中途放弃 |

### 友人帐状态

| 状态 | 中文 | 说明 |
|------|------|------|
| `active` | 活跃 | 正常显示 |
| `pending` | 待审核 | 申请中 |
| `inactive` | 失效 | 链接失效或已下架 |
